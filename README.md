## Hello Studio

Hello Studio is a playful one-page experience built with Next.js and Tailwind CSS that helps you craft the perfect “hello”. Personalize a greeting by choosing the tone, writing a name, and selecting a closing — the preview updates instantly as you tweak the inputs.

### Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Build for Production

```bash
npm run build
npm start
```

### Tech Stack

- Next.js app router with TypeScript
- Tailwind CSS with modern glassmorphism styling
- Client-side interactivity with React hooks

### Project Structure

- `src/app/page.tsx` — main page layout
- `src/components/Hero.tsx` — hero section outlining the experience
- `src/components/GreetingComposer.tsx` — interactive greeting builder
